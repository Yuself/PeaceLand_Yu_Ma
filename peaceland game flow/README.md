# Peaceland Game Flow

This folder restructures the chart prototypes into a dedicated React project.

## Intended scope

- present-first map flow
- memory flow views
- shared navigation, search, and detail panels
- data-driven node graphs
- future Google Sheet / JSON integration

## Current structure

- `src/app/`: app shell and layout
- `src/components/`: reusable graph and panel components
- `src/data/`: present and memory graph data
- `src/features/`: present and memory feature views
- `src/types/`: shared TypeScript types

## Status

This is a project restructure pass. Dependencies may still need to be installed before running the app locally.
