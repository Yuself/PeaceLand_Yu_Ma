## Practical Rule For Future Reference Searches

When asked to find a Peaceland reference:

1. Start with this file.
2. Route the request into one of these buckets: `doc/slide`, `art`, `scene`, `code`, `Yarn`, `prefab/data`.
3. Open the nearest index first, not the raw repo blindly.
4. Only after the index points to the right area, open the exact source file.

That rule should be treated as the default workflow for this workspace.
